================================================================================
API ENDPOINTS
================================================================================

GET Endpoints:
1. /                        → Landing Page
2. /dashboard               → Traffic Dashboard
3. /emergency               → Emergency Control
4. /api/traffic-data        → Current traffic status (JSON)
5. /api/history             → Traffic history (JSON)
6. /api/emergency-history   → Emergency logs (JSON)

POST Endpoints:
7. /api/detect              → Upload image detection
8. /api/detect-webcam       → Webcam detection
9. /api/emergency-override  → Emergency control
10. /api/simulate-traffic   → Generate test data

================================================================================
DATABASE TABLES
================================================================================

1. traffic_logs
   - Stores vehicle counts per lane
   - Time-stamped entries
   - Indexed for fast queries

2. emergency_logs
   - Emergency vehicle detections
   - Activation timestamps
   - Override events

3. signal_timings
   - AI-calculated timings
   - Lane-specific data
   - Historical record

4. vehicle_detections
   - Detailed vehicle types
   - Confidence scores
   - Detection timestamps

5. system_stats
   - Daily statistics
   - Peak hour analysis
   - System performance metrics
